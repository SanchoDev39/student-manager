# Student Manager (Python + Tkinter)

A desktop application for managing student records.

## Features

* Add students
* Edit marks
* Delete students
* Search students
* Sort marks
* Export to CSV
* Persistent storage using JSON

## Technologies

* Python
* Tkinter GUI
* JSON storage
* CSV export

python main.py
